# V-King-revamp
This is the new updated version of the original V-King corexy. I got permision from the original creator to do this.


# Stuff
   As i said this is not the original version.
   This is the first new version of the V-King so it will still have some problems. Because this is gonna be the cheap version, it will use a lot of an ender 3 pro parts, but you can modify it with instruction to make it how you want.
   All of the .stl files you will need are in the ST folder.
   I recomend printing everything is ABS, PETG or PCTG at atleast 13% infill.
   This version is smaller than the original but in the future i would to create another model (V-King MAX) where it would be better in every way, and a smaller model (V-King mini) that would be more easy to carry aroud.
